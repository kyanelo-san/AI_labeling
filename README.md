# 野菜販売機 在庫見守りシステム (Vegetable Vending Machine Monitor)

ご家族が運営する野菜の自動販売機の在庫状況を、自宅や外出先から簡単に、かつ直感的に確認できることを目的としたシステムです。

---

## 01. 開発の背景とコンセプト
### 「家族のための使いやすさ」を最優先に
販売機まで何度も在庫を確認しに行く手間を減らすため、家族が日常的に利用することを想定して開発しました。
専門的な知識がなくても、パッと見て「今、補充が必要か」が判断できるよう、装飾を抑えた**シンプルで簡易的なデザイン**を採用しています。

---

## 02. 主な機能と工夫点

### 📸 ハイブリッドな状況確認（右：リアルタイム / 左：写真）
技術的な確実性と分かりやすさを両立させるため、画面を左右に分割した構成にしています。
- **右側（リアルタイム）**: 現在の販売機の最新状況を表示。
- **左側（10分前の写真）**: 10分おきに撮影された静止画を表示。
- **工夫した点**: 自分の技術力で最も安定して情報を届けられる「10分に一度の撮影」をベースに、過去の画像と比較できるようにすることで、売れた形跡を視覚的に追いやすくしました。

### 🔍 在庫判別機能
現在は、各ポケットが「空いている（売り切れ）」か「入っている（在庫あり）」かを自動で判別し、一目で状況を把握できます。

---

## 03. 今後の展望（ロードマップ）
現在は在庫の有無の判別のみですが、今後は以下の機能を追加し、より便利な販売管理ツールへと進化させる予定です。

* **販売個数の自動カウント機能**
    「どの時間帯に何個売れたか」を正確に集計し、データとして蓄積できるようにします。
* **補充タイミングの最適化**
    蓄積したデータから売れ筋の時間帯を分析し、効率的な補充計画を立てられるように改善します。

---

## 04. 使用技術（システム構成）
* **言語**: Python
* **Webフレームワーク**: Flask
* **画像処理・制御**: OpenCV
* **画像保存**: Google Drive API
* **物体検出**: YOLO（将来的なカウント機能への活用を予定）
